# 安装说明

## 安装发布包

当前仓库附带的可安装文件位于：

`dist/note-batch-exporter-v3.0.2.xpi`

安装步骤：

1. 打开 Zotero
2. 进入 `工具 -> 插件`
3. 点击右上角设置按钮
4. 选择“Install Add-on From File... / 从文件安装附加组件”
5. 选择 `dist/note-batch-exporter-v3.0.2.xpi`
6. 安装完成后重启 Zotero

## 安装后入口

安装成功后，在 Zotero 主窗口顶部菜单中可以看到两个入口：

- `工具 -> 批量导出分类笔记`
- `工具 -> 导出选中条目笔记`

## 常见问题

### 工具菜单中看不到入口

- 先确认插件版本是否为 `3.0.2`
- 重启 Zotero 再检查
- 若有其他插件修改工具菜单，先临时禁用排查冲突

### 导出选中条目时没有内容

仅以下类型会被导出：

- 独立笔记
- 普通文献条目下的子笔记

附件、PDF、无笔记条目不会导出。

### 导出的 Markdown 文件重名

插件会自动在同名文件后追加 `_1`、`_2` 等后缀，避免覆盖。
