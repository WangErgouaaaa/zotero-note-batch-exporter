# Zotero Note Batch Exporter

A lightweight Zotero 8 plugin for exporting notes to Markdown.

请选择语言：

- [English Documentation](./README.en.md)
- [简体中文说明](./README.zh-CN.md)

## Quick Overview

- Export all notes under the selected collection
- Optionally include subcollections
- Export notes from currently selected items only
- Convert Zotero note HTML to Markdown
- Handle invalid filenames and duplicate filenames automatically
